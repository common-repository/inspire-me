Inspirations occur at anytime, anywhere. But you can make them appear on the top corner of your admin dashboard too. Displays over 175 bite-sized inspirations randomly.

== Description ==

Inspire Me is a tiny plugin that sits on your shoulders like a butterfly to inspire you with bite-sized quotes.

This shows up near the right side top portion of your WordPress admin screens (all screens) - exactly next to your 'Howdy, name' area.
This has more than 175 small 2 words to few words one-liners that motivate.

Here are some examples:
*   Beginnings Are Always Messy.
*   Never Doubt Your Instinct.
*   All You Have Is Today.
*   Build Your Discomfort Zone.
*   Sweat Is Just Fat Crying.

   
== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/inspire-me` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. There is no step 3. Sorry to disappoint you.
